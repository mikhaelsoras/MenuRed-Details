# MenuRed

Bem vindo ao MenuRED um sistema para automação de restaurantes.

## Funcionalidades

* Suporte a Touch e Teclado
* Controle de Estoque
  * Simples
  * Composto
* Controle de Mesa
  * Setores de Mesa
  * Filtros
* Lançamento de Produto
  * Pesquisa por Categorias e Subcategorias (Touch)
  * Pesquisa por Nome e Codigo.
  * Modificadores
    * Permite vincular valor ao modificador.
* Pagamento
  * Pagamento Parcial
  * Taxa de Serviço
  * Couvert Artístico
  * Desconto por Valor e Porcentagem
  * Identificação de Cliente
* Caixa
  * Historico de Movimentação
  * Sangria
  * Suprimento
* Emissão Fiscal
  * NFCe
  * SAT

## Detalhes Técnicos

### Servidor (Local)

O servidor foi desenvolvido em Delphi, atualiza o banco pela API online e disponibiliza uma API local para os terminais consumirem. A API foi desenvolvida com REST Dataware.

### Terminal

O terminal foi desenvolvido em Delphi utilizando FireMonkey. A camada de acesso a dados consome a API do servidor local.
